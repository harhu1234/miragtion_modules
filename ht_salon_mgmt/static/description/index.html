salon Management

salon Management for Services,Invoicing

This module has been tested in community version 14 only.


Highlights Of The Application

- Manage Customer Appointment

- Can Set Offers To our Customers

- Can Set Packages To our Customers

- Can Set Membership To our Customers

- Can configure Coupon Code To Providing Services.

- Can Sending Email For Customer Appointment

- Can print Report To During Day,Month And Date Range.

- Also can generate Report To Customer Appointment.


Complete process for configuration and functionality explained below.

  • After installation, go to salon Management, here you will see menus

  • Hear, You Can Create Appointment.




  • In Appointment Section You Can Select your Services.

  • Also you can select packages And price will be change as per the packages.




  • Can use Coupon Code For Specific Service

  • Hear, you can see price will be change as per the coupon code.




  • Can Generate Appointment Report

  • We can use state for Appointment




  • Hear we can see Service Coupon Menu

  • To Create Coupon code For provide Some Discount to our Customer Services.




  • Hear we can see Service Menu

  • To Create Service For provide Customer Services.




  • Hear we can see Membership Menu

  • To Create Membership For Customer And Apply Discount On it.




  • Hear we can see Package Menu

  • To Create Package For Customer And Apply services to particular package.

  • Also You can Apply Discount on it.




  • Hear we can see Offers Menu

  • To Create Offer For Customer And Apply services to particular Offers.

  • To Generate Offer Email to Send Our Customers.

  • Also You can Apply Discount And Date Range for offer expire.




  • Hear we can see Reporting Menu

  • To Provide Atendee And Customer Report.

  • We can see All offers,packages And Services Reports for Date,Month And Date Range.

  • Also can provide report of offers,package who can use this offer and package.




  • Hear we can see Section Menu

  • To Create section Position of Customer.

  • To select thier Seat and Floor.




  • Hear we can see Appointment Excel.

  • To Provide Selecetd Customer Report in Excel.
