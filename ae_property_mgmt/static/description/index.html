Property Management

This module will help you to manage all property related details like layout details,sales and tetancy etc.

This module has been tested in Community and Enterprise version 14 only.


Complete process for configuration and functionality explained below.


Highlights Of The Application

- You can add Landlord,buyer and tetant details.

- You can Facility and Service related details.

- You can create the tentancy agreement for tetant and create invoice for it.

- buy the property and create the invoice for it .


Complete process for configuration and functionality explained below.


Go to Property management menu >> Settings submenu and configure details about Facility and service and Layout.


For Facility and Services configuration Go to the Property management menu >> Settings >> Facility and Services submenu.



For Building Layout detail's configuration Go to the Property management menu >> Settings >> Building Layoutsubmenu.

You need to add landlord,buyer and tentant details.


To add the landlord(who sell the property) details,Go to the Property management menu >> Contacts >> Landlord submenu.





To add the tentant(Who rent the proeprty) details,Go to the Property management menu >> Contacts >> Tentant submenu.



To add the Buyer(who purchase the property) details,Go to the Property management menu >> Contacts >> Buyer submenu.

You can configure the Property related details from the property management menu >> Property submenu >> Property detailssubmenu.
