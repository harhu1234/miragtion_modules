Hide Menu or Submenu For Specific Users or Groups.

This module has been tested in Community and Enterprise version 14 only.


Features:

- This application will allows you to Hide Menu or Submenu For specific Users or Groups.


Complete process for configuration and functionality explained below.


To Hide any menu or submenu for specific user go to settings -> Technical -> Security -> Menu Access Security.



Here Select any Menu or submenu which you wants to hide.



Here Select Specific Groups for whom you want to hide a Menu.



Here Select Specific Users of that Group for whom you want to hide a Menu.



Now, here we will Hide Invoicing Menu For Administration Group Admin User.



Here You can see that Admin User is not able to see Invoicing Menu.
