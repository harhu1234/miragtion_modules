Advance Date Filter

This application will add Advance Date Filter in Search for all application

This module has been tested in Community and Enterprise version 14 only.


Complete process for configuration and functionality explained below.


To see a Advance Date Filter menu Go to settings -> User&Companies -> Groups -> click on Enable Advance Date Filter option.



Here Add a Users to whom you wants to allow to see Advance Date filter menu.



Now Go to Advance Date filter Menu and create a Record add a name, model name and field name on which you wants to add a Date filter and select a filters which you wants to add and click on confirm Button.



You can see that all selected filters are added.



If you want to add a filter based on another field you can click on Set To Draft Button and you can edit Filter field option.
