Website Product Price Invisible for public user

Now hide product price from public user.

This module has been tested in Community version 14 only.


Highlights Of The Application

- Configuration to hide products price from public user

- Very useful for multi-website platform

- Hide add to cart button for all users.


Complete process for configuration and functionality explained below.


After installation, go to Website >> Configuration >> websites, here you can see one option, Hide price, select it to hide products price from public user.



You can see logged in user can see product price



You can see public user can not see product price
