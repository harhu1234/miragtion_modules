PDF Reports Creator

This module allows user to create report templates from the GUI ,
with 20 report templates in
mail , hr , hr contract , account , sale , purchase , stock , point of sale

Create Report Template


Form this page you can create new template for one or more object on your system
To create template you have to write the template name , select the object that you want to read it's data , check apply to models if you want to create this template for more than one model , check use as sample it you want to use this template as sample template inorder to use it agine for many models , and then design your template in the text editor area

Using the text editor


To design report formate we have to use the test editor in odoo which give us many option to desing text and upload images and more , and also we can read data of one field from the object that we selected by typing
S{object.field_name}
EG : say we selected the employee name and we want to read the age of the employee which is stored in afield by name "emp_age" , so we can do it as S{object.emp_age}

Paper Format


In the template creation page you can fine paper formate page which allows you to define the paper marigns , size and orientation

Available Fields


In this page you can find all available fields from the models you selected which you can read data stored in any one of them

Samples


In this page you can find all available sample templates that you had already created and defined them as sample templates , and after selecting one of them it's text area content will be written in the current text area and then you can edit as you want

Print Report


Form create new report page you can create and print new report for one record from one model by using template that you had already created
