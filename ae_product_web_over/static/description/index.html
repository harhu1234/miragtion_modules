Product Detail Web Over

This application will allows you to see Details of product view in Saleorder, purchase order, Stock Picking, and Account invoice Based on Some Configuration.

This module has been tested in Community version 11 only.


Complete process for configuration and functionality explained below.


To enable Product Detail Quick view Go to Sales -> Configuration -> Select Sale Order Product Details Popover option.



Here In sale order lines Hover on Product You can see the product details.



To enable Product Detail Quick view Go to Purchase -> Configuration -> Select Purchase Order Product Details Popover option.



Here In Purchase order lines Hover on Product You can see the product details.



To enable Product Detail Quick view Go to Inventory -> Configuration -> Select Stock Product Details Popover option.



Here In Stock move lines Hover on Product You can see the product details.



To enable Product Detail Quick view Go to Invoicing -> Configuration -> Select Invoice Product Details Popover option.



Here In Account Invoice lines Hover on Product You can see the product details.
