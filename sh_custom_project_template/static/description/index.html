Project Custom Task Template

This module is useful to generate repeated task. First need to create template for repeated task. Select template in project and easily generate task on fire.

Key features:
  • Useful for repeated task.
  • Easy to use

Only "Project Manager" can create project template, and can create task from project by choosing template. Create project template and create task List. Assign user and set description to each task. Set Task as per template requirement.

on click of "Add an item" pop up window opens. Set Task name and select user.

Created project template list.

"Generate Task" and "Project Template" available to "Project Manager". Only "Active" templates are able to set.

First select template and then click on "Generate Task" button.

All task created inside "project.task" as per template on single click.
