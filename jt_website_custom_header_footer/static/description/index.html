Custom Header Footer For Website

Customize your website header footer with stylist theme

This module has been tested in Community version 14 only.




Highlights Of The Application

- Configuration to set customised stylist header for website

- Configuration to set customised stylist footer for website

- Configuration to add app store link (Android app link, Apple app link)

- Configuration to add different logo for header and footer for your company



Complete process for configuration and functionality explained below.



To customise header footer go to Settings -> General Settings -> Website -> Custom Header and Footer





Click to allow in custom header for website
