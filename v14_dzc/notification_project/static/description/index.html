Tasks combined in a single-email to-do list

Keep your employees informed of overpassed and today tasks

The app goal is to remind users about tasks to do them without broken deadlines:

  • All the tasks are summed up in a single list: reach them just by pushing a button in the email
  • The notification is sent early in the morning to provide a useful tool to start a day
  • Each reminder is individual and sent according to the user and the deadline indicated on a task form

Features

  • Notifications are configurable based on project tasks stages: only up-to-date tasks are sent
  • Reminder template and style are editable via standard Odoo tools (Settings > Technical > Email > Templates > "Overdue Tasks"). It supports multi languages
  • The time of each email dispatch may be changed in Odoo scheduled actions (Settings > Technical > Automation > Scheduled Actions > "Notify users about overdue tasks")
  • Full compatibility with other Odoo apps

Example

  1. John Brown is a project user and he is currently assigned to 4 tasks
  2. Tasks and deadlines are the following: 'task 1' - "Analysis: 15/11/23"; 'task 2' - "In progress: 15/11/29"; 'task 3' - 'Analysis: 15/11/30'; 'task 4' - 'Done: 15/10/11'
  3. You have configured task stages in order to send notifications with stages "Analysis", "In progress", "Testing". "Done" and "Canceled" stages are not included into the reminder
  4. Today is 15/11/29. Thus, John Brown will get the single list containing 'task 1', 'task 2' with stated deadlines
