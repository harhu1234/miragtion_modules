Odoo Facebook Messenger Chat

Reply to your customers query instantly!

Key features:

Facebook Messenger Chat to Odoo Website

Chat with Customers Via Facebook Messenger

Greeting Templates from Odoo Backend


FEATURES LIST

Enable/disable Facebook Messenger Chat

  • Enable/disable facebook messenger chat checkbox in Odoo Website

Customers can Easily connect to you

  • Customers need to login to their Facebook account to connect to you.

Reply to Customers Via Facebook Messenger

  • Get the customer's message in Facebook Messenger Inbox.
  • Reply to your customer's message directly from Facebook.

Set Theme Color & Template from Odoo backend

  • Set custom color theme code in Odoo backend setting.
  • Add Logged in and Logged out Greeting Message from backend.

Facebook Chat Widget in Odoo Website

  • Enable Messenger checkbox to configure Facebook Messenger Chat with your Odoo Website
  • You will get the following options to fill; once, you will enable the checkbox.
  • Add Facebook Page Id which is available in the Facebook page.
  • Add Messenger Theme Color Code.
  • Fill Logged in and Logged out Greeting Message.
  • Log in to your Facebook account and create a Facebook page.
  • Go to the About section to get the Facebook Page Id.
  • Go to the Advanced Messaging of the Facebook page settings
  • Add the Website domains of your Odoo database you want to configure with Facebook messenger chat
  • Log in as a customer to Odoo Website and you can see a Facebook chat widget.
  • Clickable facebook chat widget in Odoo Website
  • Customer needs to continue with their Facebook account to chat
  • Customers can chat with you hassle-freely
  • Customers can select different actions directly from the chat
